# genre-bender

Cross-pollinate genres, generate hybrid sonic identities, and build Suno/Udio prompts from the fusion.

Every genre is a toolkit. Every toolkit can be stolen.
This command maps the collision points between worlds and turns them into production blueprints.

## Context Needed

Ask for:
- Source genre(s): what you already make / where you're coming from
- Target genre(s) or reference artists: what you want to pull from
- Intended output: production prompt / track concept / new sound identity / full brief
- Tempo range if any preference
- Mood/energy target
- Whether this is for AI generation (Suno/Udio) or live production in DAW

## Process

1. **Deconstruct each genre** into its atomic components:
   - Rhythm pattern (kick placement, snare placement, hi-hat feel)
   - Harmonic vocabulary (chord types, progressions, dissonance tolerance)
   - Timbral signature (synth vs acoustic vs electronic, texture density)
   - Arrangement conventions (verse/chorus, drop structure, call-and-response)
   - Tempo and time feel (swing, quantize level, groove pocket)
   - Reference production techniques (sidechain culture, reverb philosophy, dynamic range)

2. **Map the collision points** — where do the genres naturally meet or violently clash?
   - Natural overlaps: rhythmic feel, shared BPM ranges, emotional territory
   - Productive tensions: one genre's approach applied to another's instrument
   - Ironic juxtapositions: content/feel mismatch that creates something new

3. **Design the hybrid** — define the blend:
   - What percentage of each genre? (60% X / 40% Y is a recipe, not a vibe)
   - Which elements come from which genre (kick from A, harmony from B, texture from C)
   - What makes this fusion novel vs just a mashup?

4. **Name the new sound** — every genre-bender needs a handle for recall and pitching

5. **Generate production blueprint** — specific, actionable for DAW or AI generator

## Rules

- Deconstruct before blending — shallow genre knowledge makes shallow fusions
- Name the new sound: unnamed blends don't stick, in your catalog or in pitches
- The ironic juxtaposition (heavy metal drums under jazz harmony) is often more interesting than the smooth blend
- Suno/Udio prompts: specificity wins over genre names alone — describe textures and feels, not just labels
- If the fusion has been done before, say so and name the existing subgenre — then push past it
- "70% X / 30% Y" is a starting point; the blend ratio shifts track by track
