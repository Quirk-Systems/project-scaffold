# thirst-traps

Engineer social content that pulls people in before they know why.

Posting is not marketing. Posting is fishing.
Every piece of content is either bait, a hook, or filler.
This command builds the bait, sets the hook, cuts the filler.

## Context Needed

Ask for:
- Release type: single / EP / album / merch / show / collab / just building
- Platforms: Instagram / TikTok / X / YouTube / Threads / Facebook / all
- Release timeline: drop date + runway available (days/weeks)
- Current audience size and engagement level (context for what's realistic)
- Assets available: stems / acapella / video / artwork / BTS footage / none yet
- Brand voice: how does this artist show up? (cryptic / hype / vulnerable / funny / mysterious)
- Budget for ads: $0 / minimal / real spend

## Process

1. **Release calendar architecture** — map content to the release arc:
   - Tease phase (2-4 weeks out): build curiosity without revealing
   - Build phase (1-2 weeks out): social proof, context, stakes
   - Drop phase (48hrs): activation, push, share-ask
   - Sustain phase (post-drop): extend the moment, playlist pushes, re-engagement

2. **Content type matrix** per platform:
   - TikTok: hook in frame 1 / vertical / trend-leverage / duet/stitch bait
   - Instagram: Reels > carousel > static / Story as behind-the-scenes / link in bio strategy
   - X/Twitter: hot take → discourse → quote replies / timing to trends
   - YouTube: Shorts for discovery / full video for depth / Community posts for engagement

3. **Hook engineering** — first 3 seconds / first line rule:
   - Pattern interrupt (unexpected visual, sound, or statement)
   - Open loop (question or incomplete thought that demands resolution)
   - Immediate value signal (tell them why to stay)

4. **Batch the content** — write captions, scripts, and copy for each phase

5. **Engagement traps** — design content that generates comments, saves, shares:
   - Poll / question / "comment X if you..." mechanics
   - Debate bait (strong opinion that's shareable even in disagreement)
   - Save-worthy (reference, checklist, visual that people want to return to)

## Rules

- Never post without a purpose — every piece of content should do one job (curiosity / click / share / save)
- The platform is the medium — don't post Instagram content to TikTok
- Authenticity is a strategy, not an accident — being "real" on purpose still requires craft
- Comments > likes — optimize for conversation starters
- Never beg for streams — create reasons to listen instead
- The post-drop sustain phase is where 80% of artists go silent; plan it in advance
- Strong opinion + specific audience beats broad appeal every time
